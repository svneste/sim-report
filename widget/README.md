# amoCRM Widget — Отчёт по сим-картам

Тонкая обёртка над React SPA, открывает фронт в iframe внутри amoCRM.

## Сборка

```bash
cd widget
zip -r ../sim_report_widget.zip manifest.json script.js i18n images
```

## Установка в amoCRM

1. Настройки → Интеграции → «+ Создать интеграцию» → «Виджет для amoCRM».
2. Загрузить `sim_report_widget.zip`.
3. После установки виджет появится в Настройках. Включить.
4. В левом меню (или в настройках) появится пункт «Отчёт по сим-картам».

## Конфигурация URL фронта

Перед сборкой замените `FRONT_URL` в `script.js` на адрес развёрнутого фронта.
В деве это `http://localhost:5173`, в проде — публичный https-домен.

## Иконки

Положите в `images/`:
- `logo.png` (50×50)
- `logo_small.png` (30×30)
- `logo_dark.png` (50×50, для тёмной темы)
